# Minimal Template

A **bare-bones** CLAUDE.md template. Use this as a starting point for any project type.

## What's included
- Essential sections with placeholder comments
- No framework-specific assumptions
- Quick to fill in (under 5 minutes)

## Usage
```bash
cp templates/minimal/CLAUDE.md your-project/CLAUDE.md
```

Fill in the bracketed placeholders with your project's details.

## Best for
- Quick prototypes
- Unfamiliar or niche stacks
- When other templates don't fit your setup
