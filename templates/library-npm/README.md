# npm Library Template

CLAUDE.md template for **npm packages** and libraries.

## What's included
- Dual ESM + CJS build setup (tsup)
- Public API design principles
- Package.json exports configuration
- Publishing checklist
- Testing strategy for libraries

## Usage
```bash
cp templates/library-npm/CLAUDE.md your-project/CLAUDE.md
```

## Best for
- Open source npm packages
- Internal shared libraries
- CLI tools
- Utility libraries
