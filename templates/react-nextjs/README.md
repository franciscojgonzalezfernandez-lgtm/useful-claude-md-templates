# React / Next.js Template

CLAUDE.md template for **Next.js 14+ (App Router)** projects with TypeScript.

## What's included
- App Router conventions (Server Components, Server Actions, route groups)
- Tailwind CSS + component patterns
- Prisma + PostgreSQL database layer
- Testing setup (Vitest + RTL + Playwright)
- Performance and security guidelines

## Usage
```bash
cp templates/react-nextjs/CLAUDE.md your-project/CLAUDE.md
```

Edit the `[Project name]` and `[brief description]` placeholders, then customize sections to match your specific setup.

## Best for
- SaaS applications
- Marketing sites with dynamic features
- Full-stack Next.js apps with database
- Projects using Vercel deployment
