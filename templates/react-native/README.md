# React Native / Expo Template

CLAUDE.md template for **React Native** projects using Expo.

## What's included
- Expo Router (file-based navigation)
- Mobile-specific patterns (lists, keyboard, safe areas, offline)
- NativeWind / StyleSheet styling conventions
- Performance guidelines for mobile
- Testing with RNTL and Detox

## Usage
```bash
cp templates/react-native/CLAUDE.md your-project/CLAUDE.md
```

## Best for
- Cross-platform mobile apps (iOS + Android)
- Expo managed workflow projects
- Apps with offline-first requirements
