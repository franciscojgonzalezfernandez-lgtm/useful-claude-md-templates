# NestJS Backend Template

CLAUDE.md template for **NestJS** backend projects with Clean Architecture.

## What's included
- Clean Architecture layer structure (domain → application → infrastructure → presentation)
- TypeORM/Prisma database patterns
- JWT authentication setup
- Swagger API documentation conventions
- Error handling and validation patterns

## Usage
```bash
cp templates/nestjs-backend/CLAUDE.md your-project/CLAUDE.md
```

Edit the `[Project name]` and `[brief description]` placeholders, then adjust the architecture sections to match your project.

## Best for
- REST API backends
- Microservices
- Enterprise applications with complex business logic
- Projects following Domain-Driven Design
